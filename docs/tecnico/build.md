# Build do RadioRoots

Este documento é para desenvolvedores que desejam compilar o jogo a partir do código-fonte.

---

## Requisitos

- Clone o repositório ```git clone https://github.com/Jovito5s9/Jogo```
- Certifique-se de ter Python 3.10+ em sua maquina
- Instale as dependências ```pip install -r requirements.txt```, pode ser preciso estar em uma venv
- Linux recomendado

---

### Build para Linux

Execute o script ```build.sh``` e procure o jogo em dist/RadioRoots

### Build para Windows

Execute o script ```buildWindows.bat``` e procure o jogo em dist/RadioRoots

### Build para android

Execute pelo terminal na raiz do projeto ```buildozer android debug``` e procure pelo jogo em /bin
